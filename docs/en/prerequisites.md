# Prerequisites

## Required

- Node.js 18+ for the unified installer
- Python 3.10+ for the Scibudy runtime
- internet access for provider APIs and package/model downloads

## Recommended

- Codex installed if you want MCP integration
- Git for source installs

## Optional

- conda, Linux, and NVIDIA GPU for the full local semantic retrieval path

## You do **not** need all of these for the base profile

For `base` and most first-time installs, you only need:

- Node.js
- Python

GPU, conda, and local large-model setup are only needed for `gpu-local` or `full`.
