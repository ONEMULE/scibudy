# Scibudy

Scibudy is a Codex-native scientific research expansion assistant.

Scibudy 提供面向 Codex 的学术检索、文献库管理、全文分析与本地语义检索能力。

## Choose your language

- [English documentation](en/index.md)
- [中文文档](zh/index.md)

## Quick start

```bash
npx scibudy-install --profile base
```

## Common next steps

- `scibudy doctor`
- `scibudy search "your topic"`
- `scibudy ui --open`
- `scibudy install-codex`

## Local GPU option

For Linux + NVIDIA systems:

```bash
scibudy install-local-models
scibudy warm-local-models --background
```
